# 🎮 Simon Game - Memory Challenge

## 📌 Description
The **Simon Game** is a fun and interactive memory-based game that tests your ability to recall sequences. The game starts by flashing a color, and the player must click the correct button. With each level, a new color is added to the sequence, making it progressively harder!

[🎮 Play the Game Here](https://67eacd8fa0e933a4668f9097--effortless-dodol-b3eacd.netlify.app/)

---

## 🎯 How to Play
1. **Press any key** to start the game.
2. Watch carefully as a button flashes.
3. Click the buttons in the same order as they flashed.
4. The sequence gets longer with each level.
5. If you click the wrong button, **Game Over!** Press any key to restart.

---

## 🛠 Features
✅ **Dynamic Sequence Generation** – A new color is added each round.
✅ **Visual and Interactive Feedback** – Buttons flash to indicate the sequence.
✅ **Progressive Difficulty** – The sequence grows longer as you progress.
✅ **Game Over Detection** – Incorrect input resets the game.
✅ **Keyboard and Click Support** – Press any key to start, then use mouse clicks.
